Aprende el uso del arduino con proyectos sencillos y rapidos de aprender

Crea un semaforo

Un proyecto de Arduino para un semáforo es como hacer un juguete que controla las luces de la calle. Puede mostrar rojo para detenerse, verde para avanzar y amarillo para estar listo. Es como tener un robot pequeño que ayuda a las personas a cruzar la calle de manera segura

El sensor de distanciamiento social Arduino

El sensor de distanciamiento social Arduino es como un ojo mágico que ayuda a un amigo robot llamado Roby a mantenerse seguro. Puede ver cuán lejos están las cosas y si Roby se acerca demasiado, le avisa para que retroceda y mantenga una distancia segura. Es como un superpoder que lo protege mientras juega.

Ruleta de numeros

Imagina que un robot te da numeros al azar con este proyecto lo puedes llevar a la realidad

Boton Led Arduino

Crea un circuito donde al apretar un boton prendas y apagues un led, suena sencillo verdad!!!

Calculadora

Crea un circuito similar a una calculadora, podras hacer operaciones sencillas como sumas, restas, multiplicacion y division de 2 cantidades
